Positioning

Floating might be good, but, what if I don't want my object attached to a side?

This little fucker here has somewhat a 200px "invisible border" to the left

In order to do that, we'll use POSITIONING!

Positions can also be ABSOLUTE, meaning they will ignore other elements in the same place.

But these absolute ones can be moved with "left" and "top" comands too.

Absolute positioning takes the element OUT OF THE FLOW OF THE PAGE. IT MAKES IT INDEPENDENT!

Other type is FIXED. FIXED DOESNT MOVE WHEN THE PAGE IS SCROLLED

We can set the opacity of an element as well!
